## Run Locally

Clone the project

```bash
  git clone https://github.com/Gustavo19951/Book-App.git
```

Go to the project directory

```bash
  cd Book-App
```

### Front End

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Deployment

To deploy this project run

```bash
  npm run build
```
