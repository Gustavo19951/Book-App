## Run Locally

Clone the project

```bash
  git clone https://github.com/Gustavo19951/Sistema_Integrado.git
```

Go to the project directory

```bash
  cd Sistema_Integrado
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
